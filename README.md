# CLI Text Utilities
A collection of command-line utilities for text processing and manipulation.
- Count characters
- Count vowels
- Detect uppercase/lowercase
- Classify password strength
  - Minimum length
  - Contains number?
  - Contains uppercase letter?
  - Classify as weak / medium / strong
- Reverse text

## Milestone plan
## 1: One-file version
Start with:
- Menu
- Basic functionalities

## 2: Add loops and validation
- Make the menu repeat until exit
- Reject invalid options cleanly

## 3: Extract functions
- Move every feature into its own functions.

## 4: Split into packages
- Refactor calculations and validations into separate packages.
